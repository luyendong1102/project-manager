API Document

Mọi API đều bắt đầu từ tiền tố /api/v1/*

Ví dụ : https://host.net/api/v1/user/register

Các token để xác định danh tính được lưu trong header với thuộc tính 'key'

Các token được khởi tạo mới khi login và có thời hạn trong vòng 3 ngày kể từ khi nó được tạo ra, Server không có trách nhiệm lưu lại token khi nó được tạo, token có giá trị như khóa, không được để lộ token, khi muốn đăng xuất hãy xóa token được nhận. Server là stateless vì thế không có session cho mỗi lần sử dụng

Authentication

POST EveryOne /api/auth

Mỗi một tài khoản khi tạo sẽ có một username riêng biệt và password. "username" và "password" là thuộc tính được gửi khi muốn đăng nhập, phương thúc : POST

User

GET QUANLY /user/all

Lấy toàn bộ danh sách nhân viên. Chỉ tài khoản có role QUANLY mới có quyền truy cập

GET QUANLY /user/upgrage/{id}

Nâng cấp tài khoản có id {id} lên QUANLY. Chỉ có tài khoản role QUANLY mới có quyền truy cập

POST EveryOne /user/register

Đăng ký tài khoản mới. Ai cũng có thể truy cập

GET EveryOne /user

Xem thông tin tài khoản của mình. Chỉ chủ tài khoản mới có quyền truy cập

POST EveryOne /user/changepassword

Đổi mật khẩu. Chỉ chủ tài khoản mới có quyền truy cập

GET EveryOne /user/disable

Hủy kích hoạt tài khoản, liên hệ luyendong để mở lại. Chỉ chủ tài khoản mới có quyền truy cập

GET EveryOne /user/find?p=

Tìm thông tin tài khoản với tham số p. Ai cũng có thể truy cập

Project

GET QUANLY /project

Trả về thông tin của tất cả project mà tài khoản quản lý

GET QUANLY /project/{id}

Trả về thông của project có id mà tài khoản quản lý

GET QUANLY /project/find?p=

Tim project mà chưa tham số p

POST QUANLY /

Tạo mới project hoặc chỉnh sửa với người quản lý là người tạo, Yêu cầu object Project

DELETE QUANLY /project/{id}

Xóa project với người quản lý

Task

POST QUANLY /task/create

Tạo mới task

PUT QUANLY /task/update

Cập nhật thông tin task

POST QUANLY /task/delete/{id}

Xóa task chỉ chủ task

GET QUANLY /task/project/{id}

Lấy tất cả các task từ project mình quản lý

GET EveryOne /task/self

Các task của mình

GET EveryOne /task/single/{id}

Lấy ra một task có {id} mà liên quan đến mình

GET EveryOne /task/find

ERROR

POST QUANLY /task/assign/{id}

Thêm người vào task vào {id} nhận request body là User

POST QUANLY /task/remove/{id}

Xóa người vào task vào {id} nhận request body là User
